# 🎵 Music Store Data Analysis using SQL

An end-to-end **SQL Data Analysis** project that explores a digital music store database to answer real-world business questions and generate actionable insights using SQL.

This project demonstrates how SQL can be used to analyze customer behavior, sales performance, product trends, and business operations through relational databases.

# 📖 Project Overview

Businesses generate large amounts of transactional data every day. This project demonstrates how SQL can be used to extract meaningful insights from that data to support business decision-making.

Using a digital music store database, various SQL queries were written to answer real-world business questions related to customers, sales, artists, genres, invoices, and employee performance.

The project covers SQL concepts from beginner to advanced level and showcases how relational databases can be used for analytical reporting.

---

# 🎯 Objectives

- Analyze customer purchasing behavior
- Identify top-selling artists and genres
- Discover revenue trends
- Evaluate employee performance
- Practice SQL using real-world business scenarios
- Strengthen analytical and problem-solving skills

---

# 📊 Dataset Information

The dataset represents a digital music store and contains multiple related tables, including:

- Customers
- Employees
- Artists
- Albums
- Tracks
- Genres
- Invoices
- Invoice Items
- Media Types
- Playlists

The database follows a relational model, making it suitable for practicing joins, aggregations, subqueries, Common Table Expressions (CTEs), and Window Functions.

# ❓ Business Questions Solved

This project answers several business questions, including:

- Who is the senior-most employee?
- Which countries have the highest number of invoices?
- What are the top-performing cities based on revenue?
- Who are the highest spending customers?
- Which music genres are the most popular?
- Which artists generated the highest revenue?
- Which customers spend the most on music purchases?
- Which country generates the highest sales?

---

# 📈 Key Business Insights

- Revenue is concentrated in a few countries and cities.
- A small percentage of customers contribute a significant share of sales.
- Certain artists and genres consistently outperform others.
- SQL enables efficient extraction of valuable business insights from transactional databases.
